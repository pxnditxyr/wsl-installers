# Win32Yank linker

*Link Win32Yank for wsl share clipboard*
- **Only**
  link win32yank of ```~/.config/nvim```

*Dependencies*
- Win32Yank

## Run or Execute | After of clone repository

**First**
```bash
cp /mnt/path-file.exe ~/.config/nvim/
```

**Second**

```bash
./win32yank-linker/link-win32yank.sh
```

**or**

```bash
cd win32yank-linker
./install-providers-nvim.sh
```
